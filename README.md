# Python Nmap Network Scanner

## Overview
This project demonstrates how to automate network scanning using Python and Nmap. 
The script performs port scanning and service detection to identify potential 
security risks in a network.

## Tools & Technologies
- Python
- Nmap
- python-nmap library
- Kali Linux / Linux terminal

## Features
- Automated Nmap scanning using Python
- Port discovery
- Service version detection
- Scan result output for analysis

## Example Command
python scanner.py <target-ip>

## Skills Demonstrated
- Network reconnaissance
- Security automation using Python
- Port scanning and service enumeration
- Cybersecurity analysis
